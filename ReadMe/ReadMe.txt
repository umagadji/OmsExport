Информация по приложению:

1. Приложение написано 12.04.2023 года на JavaFX + Spring Boot
2. Для работы приложения или доработки необходимы:
	2.1. Java 17 и JavaFX 19.0.2.1
	2.2. Установленная БД Posgtres со схемой tfoms, пользователем posgtre и паролем admin (ну или поменять как хотите)
	2.3. В spring boot приложении в связке с JAVA FX не используется модульный тип приложения
3. Приложение создаем ASUM файлы для ARMs по отделениям РДЦ (диагностика), РЭЦ, КП, Дневные стационары.
